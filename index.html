Project 2

Problem Statement

Moving past basic mock-ups, language ecosystems lack modern real-time cloud data architectures, reactive edge-validation processing engines, and hardware device layer adaptation workflows needed to deliver immediate contextual study feedback loops.

GitHub Repository YouTube VSR

Lessons Learned

Constructing this tier-1 application required handling sophisticated mobile architectures. Integrated Firebase Authentication alongside Firestore NoSQL backend configurations to handle real-time student document syncing globally. Coupled local Room SQLite engines with asynchronous flow routines to implement reliable offline-first functionalities. Programmed Retrofit REST interfaces with OkHttp clients to query remote semantic libraries, overlaying real-world spatial contextual maps using the Google Maps SDK. Applied on-device edge AI routines via Google ML Kit and lifecycle-aligned CameraX capture pipelines to support visual object translation queries. The implementation of decoupled MVVM patterns, Hilt-driven dependency injection, and coroutine workers allowed for modular scalability while balancing thread workloads, database query constraints, and hardware lifecycle boundaries.
